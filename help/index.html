도움말

"배차" 버튼을 누르면 로그인 화면이 나타납니다. 우진교통 인트라넷 아이디, 비밀번호를 입력하면 됩니다.

로그인 한 다음에 원하는 일자를 선택하면 해당 일자 배차일지 데이터를 볼 수 있습니다. 데이터는 09:00~09:30 10분 간격, 이외 매시 정각 갱신됩니다.

화면 위쪽 검색창에 차량번호나 운전자 이름을 입력하면 해당 차량으로 화면이 이동되고 푸른색으로 표시됩니다. 5018호는 "18"이라고 써도 검색 가능합니다.

노선번호를 누르면 노선번호 목록이 표시되고, 특정 노선번호를 누르면 해당 노선으로 화면이 이동됩니다.

특정 차량을 누르면 해당 차량에 대한 정보가 표시됩니다.

간혹 오류 등으로 인해 실제 배차표와 차이가 있을 수도 있습니다.
