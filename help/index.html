로그인

"배차" 버튼을 누르면 로그인 화면이 나타납니다. 인트라넷 아이디, 비밀번호를 입력하면 됩니다.

"설정→자동 로그인"이 활성화 된 경우, 이후 로그인 과정이 생략됩니다. 자세히는 "설정" 단락을 참고해주세요.


일일 배차 확인

원하는 일자를 선택해서 해당 일자 배차일지 데이터를 볼 수 있습니다. 데이터는 09:00~12:00 10분 간격, 이외 매시 정각 갱신됩니다.

화면 위쪽 검색창에 차량번호나 운전자 이름을 입력하면 해당 차량으로 화면이 이동되고 푸른색으로 표시됩니다. 5018호는 "18"이라고 써도 검색 가능합니다.

노선번호를 누르면 노선번호 목록이 표시되고, 특정 노선번호를 누르면 해당 위치로 화면이 이동됩니다.

특정 차량을 누르면 해당 차량에 대한 정보가 표시됩니다.

배차 정보는 자동으로 갱신되지만, 차종, 주차위치 등 차량 정보는 수동으로 갱신하기 때문에 실제 현황과 일치하지 않는 정보가 표시될 수도 있습니다.


월간 배차이력 확인

"월간 배차이력" 버튼을 누르면 한달 단위로 특정 운전자 및 차량의 배차 기록을 확인할 수 있습니다.

"←", "→" 버튼을 통해 각각 이전 달, 다음 달로 페이지를 넘길 수 있습니다.

  • 운전자로 조회했을 때: ■오전 ■오후 ／휴무를 나타냅니다.
  • 차량번호로 조회했을 때: ■ 대운자로 편성된 것을 나타냅니다.
배차일지를 기반으로 표시하기 때문에, 배차표에 반영되지 않은 대리근무, 대차 등은 반영되지 않습니다.


설정

"설정" 버튼을 누르면 설정 창이 나타납니다.

자동 검색:

  • "✎_"을 눌러서 이름이나 차량번호를 설정할 수 있습니다.
  • 설정된 경우, 로그인 후에 해당 운전자나 차량이 가장 먼저 보이게 됩니다.

자동 로그인:

  • 활성화 "☑"되어 있으면 아래 조건에 따라 로그인이 유지됩니다.
  • 30일동안 접속하지 않으면 로그인이 해제됩니다.
  • 다른 기기에서 로그인하면 기존 기기에서의 로그인은 해제됩니다.

푸시알림 수신:

  • "홈 화면에 추가"된 상태에서만 설정할 수 있는 기능입니다. 상세한 내용은 하단 "홈 화면에 추가" 단락을 참고해 주세요.

  • 활성화 "☑"되어 있으면 배차표가 갱신될 때 알림을 수신받을 수 있습니다.

  • 설정 할 때 알림 권한을 허용해야 합니다.


홈 화면에 추가

휴대폰에서 "홈 화면에 추가" 기능을 사용하면 앱을 설치한 것처럼 바로 배차열람을 할 수 있고, 배차표가 갱신될 때 푸시알림을 받을 수도 있습니다. 이 기능은 삼성 인터넷, Chrome, Safari 브라우저 등에서 사용할 수 있습니다.

설치 방법:


  • 삼성 인터넷

  • Chrome

  • Safari
